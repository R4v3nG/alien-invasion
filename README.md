# alien-invasion
My version of Alien Invasion game from book.


## Installation

```bash
apt install python3-pip
git clone https://github.com/R4v3nG/alien-invasion.git
cd alien-invasion/
pip3 install -r requirements.txt
python3 alien_invasion.py
```
